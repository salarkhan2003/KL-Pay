🎓 About KL ONE

Built for
KL University

KL ONE is the official campus super-app — designed to make student life simpler, faster, and more rewarding.

Our Mission

Simplifying
Campus Life

KL ONE was built to solve a real problem: students wasting time in queues, merchants struggling with cash management, and no unified way to pay on campus.

We built a platform that connects every student, merchant, and administrator on campus through a single, beautiful app — with real-time ordering, instant UPI payments, and a rewards system that makes every purchase count.

Tech Stack

Built with modern, production-grade technology.

⚛️

React + TypeScript

Frontend built with React 19 and TypeScript for type-safe, maintainable code.

🔥

Firebase

Firebase Auth for secure login and Firestore for real-time data sync across all users.

⚡

Supabase

PostgreSQL database with row-level security, real-time subscriptions, and instant APIs.

💳

Cashfree Payments

PCI-DSS compliant payment gateway for UPI split payments and instant settlements.

▲

Vercel

Deployed on Vercel with serverless API functions for zero-maintenance infrastructure.

🎨

Tailwind CSS + Motion

Utility-first styling with smooth animations powered by Motion (Framer Motion).
